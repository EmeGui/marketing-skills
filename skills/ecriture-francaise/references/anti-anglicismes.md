# Anti-anglicismes — Liste de référence

> Remplacements recommandés pour les anglicismes les plus fréquents en marketing B2B français.

## Verbes et locutions verbales

| Anglicisme | Remplacement français |
|---|---|
| adresser un problème | traiter un problème, résoudre un problème |
| implémenter | mettre en œuvre, déployer, appliquer |
| monitorer | surveiller, suivre, piloter, superviser |
| prioriser | donner la priorité à, classer par priorité |
| processer | traiter |
| setup | configuration, paramétrage, mise en place |
| runner (un test, un script) | exécuter, lancer |
| forwarder (un email) | transférer |
| switcher | basculer, passer à |
| matcher | correspondre à, être en adéquation avec |

## Adjectifs et adverbes

| Anglicisme | Remplacement français |
|---|---|
| digital | numérique |
| scalable | évolutif, qui peut monter en charge, dimensionnable |
| user-friendly | facile à utiliser, accessible, ergonomique |
| data-driven | fondé sur les données, piloté par la donnée |
| cost-effective | rentable, économique, efficace |
| seamless | fluide, transparent |
| real-time | en temps réel (acceptable, l'usage est installé) |
| cross-platform | multiplateforme, interplateforme |

## Noms

| Anglicisme | Remplacement français |
|---|---|
| use case | cas d'usage (acceptable), cas d'utilisation |
| delivery | livraison, restitution, réalisation |
| insight | enseignement, éclairage, analyse, piste |
| funnel (marketing) | entonnoir (acceptable), tunnel de conversion, parcours |
| lead | prospect, contact |
| slide | diapositive, diapo |
| template | modèle, gabarit |
| dashboard | tableau de bord |
| workflow | processus, flux de travail, enchaînement |
| feedback | retour, commentaire, avis |
| meeting | réunion, rendez-vous |
| call | appel, visioconférence |
| report | rapport, bilan, compte rendu |
| CEO / CTO / CMO | PDG / DSI / directeur marketing, selon le titre français consacré |

## Faux-amis

| Mot anglais | Sens en français | Ne pas confondre avec |
|---|---|---|
| actually | en réalité | "actuellement" (= currently) |
| eventually | finalement | "éventuellement" (= possibly) |
| comprehensive | complet, exhaustif | "compréhensif" (= understanding) |
| consistent | cohérent | "consistant" (= thick, solid) |
| dramatic | spectaculaire, important | "dramatique" (= tragic) |
| sensible | raisonnable, sensé | "sensible" (= sensitive) |

## Règles souples

- Depuis 1994 (loi Toubon), les termes français officiels existent pour beaucoup d'anglicismes. Le site [FranceTerme](https://www.culture.fr/franceterme) est la référence.
- En B2B technique, certains anglicismes sont passés dans l'usage et leur remplacement sonnerait artificiel : "cloud", "API", "data", "pipeline". Ne pas forcer un terme français si le terme anglais est le standard de l'industrie.
- L'objectif n'est pas le purisme : c'est d'éviter le français d'aéroport, où 30% des mots sont anglais sans raison.
